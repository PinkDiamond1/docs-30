Dev Tools

Ripple provides a set of developer tools to help you test, explore, and validate XAG Ledger API requests and behavior.

  • XAG Ledger RPC Tool

    Use this tool to print raw information about a XAG Ledger account, transaction, or ledger.

  • WebSocket API Tool

    Use this tool to send sample requests and get responses from the rippled API. No setup required.

    • XAG Ledger Explorer

    View validations of new ledger versions in real-time, chart the geographic location of servers in the XAG Ledger network, and explore the state of the ledger.
